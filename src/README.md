## 🚀 Quick Start

### Installation

```bash
# Clone repository
git clone <repository-url>
cd sacred-geometry-context-tree

# Install Python dependencies
pip install -r requirements.txt

# Create workspace context tree
python -m src.core.sacred_geometry_context_tree
```

### Basic Usage

```python
from src.core.sacred_geometry_context_tree import SacredContextTree, create_workspace_context_tree

# Create context tree for workspace
tree = create_workspace_context_tree("/path/to/workspace")

# Find harmonic path between contexts
path = tree.find_harmonic_path("start_node", "end_node")

# Navigate using sacred geometry principles
tree.navigate_to("target_context", navigation_type="harmonic")

# Analyze sacred patterns
analyzer = AdvancedSacredGeometryAnalyzer(tree)
clusters = analyzer.analyze_harmonic_clusters()
pathways = analyzer.discover_sacred_pathways()
```

### Interactive Navigation

```bash
# Terminal-based navigation
python src/navigation/sacred_navigator.py

# Demo mode for automated environments
python src/navigation/sacred_navigator.py --demo

# Web-based visualization
cd src/gui && python -m http.server 8080
# Open http://localhost:8080
```
